## What is properties?

-   neo4j does not support nested properties
-   Key-Value pairs on a Node / Relationship

![plot](./images/properties.png)

<br>

Supported Value Types

-   Boolean
-   Text
-   Numbers (123, 56.70)
-   Point (2D, 3D, Lat Lon)
-   Temporal (Date, Time, DateTime)
-   Lists (list items must all be same type)
